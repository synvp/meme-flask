# Meme generator using Flask
pulling memes every 30 seconds from Reddit.

# Installation for Debian/Ubuntu:

  pip3 install flask 
  
  pip3 install requests
  
  
  
  
Clone the repo:

  git clone https://github.com/synvp/meme-flask.git
  
  
  
  
Then:

 cd meme-flask
 
 python3 meme-flask.py
 
 
 
 
And to view:

  On your browser go to http://127.0.0.1:8000
